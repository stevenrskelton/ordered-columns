Getting Started

Setup

  1. Add the library using the Javascript package manager Bower: bower install --save ordered-columns
  2. Import Web Components' polyfill: <script src="bower_components/webcomponentsjs/webcomponents.min.js"></script>
  3. Import Custom Element: <link rel="import" href="bower_components/ordered-columns/ordered-columns.html">

  4. Start using it!

    Set the column count, and add content:

Element Attributes

Attribute Options Default Description
count integer 1 Number of columns to fill
noRedraw boolean false Do not recalculate position of all elements on window.resize
columnWidth integer null Set columns to a fixed width. count will be calculated to maximally fill available width.
